# Midnight Theme for Standard Notes

Dark midnight theme plugin for Standard Notes.

This is a fork of the [official Midnight theme](https://github.com/standardnotes/midnight-theme) packaged as a free, installable plugin. The original theme is only available as a paid feature in Standard Notes, but this fork makes it freely available to everyone.

## Install the plugin

```
https://iamanaws.github.io/sn-midnight-theme/dev.iamanaws.sn-midnight-theme.json
```

Or install via the plugin URL in Standard Notes:
1. Go to **Preferences** → **General** → **Advanced Settings**
2. Click **Manage Plugins**
3. Click **Install Custom Plugin**
4. Enter the URL above
